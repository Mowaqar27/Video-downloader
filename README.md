# Social Video Downloader

A simple web app to download:

- Instagram Reels
- YouTube videos
- Pinterest videos

It includes a **paste link button** so users can quickly paste copied links.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Install `yt-dlp` on your machine/server and ensure it is available in `PATH`.

3. Start the app:

   ```bash
   npm start
   ```

4. Open `http://localhost:3000`.

## Notes

- Some private/restricted content may fail to download.
- Please comply with each platform's terms and local laws.
